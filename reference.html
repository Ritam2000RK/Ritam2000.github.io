Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tfoot> Table Foot The set of rows defining the footer in a table.
<tr> Table Row The table row container.
<th> Table header Defines header cell in HTML
<td> Table Data The table row container.

Tag Name Description

<img> Image This tag is used to insert image into web application
<ul> Unordered List To list something in unordered manner.
<ol> Ordered List To list something in ordered manner.
<form> Form This tag is used to create forms.

CSS Selectors

Tag Name Description
#example ID Selector An ID selector in HTML is a way to target a specific element using a unique ID attribute, defined in the HTML code using the id attribute.
.example CLASS Selector A class selector in HTML is a way to target an element using a specific class name, defined in the HTML code using the class attribute.
* UNIVERSAL Selector A universal selector in HTML is a CSS selector that targets all elements on a web page regardless of their type, and is represented by an asterisk (*) symbol in CSS code.
